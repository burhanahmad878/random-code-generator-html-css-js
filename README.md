# QR Code Generator

## 📌 Overview
This is a simple QR Code Generator web application built using HTML, CSS, and JavaScript.  
It allows users to generate QR codes instantly by entering text or a URL.

---

## 🚀 Features
- Generate QR codes from text or URLs
- Real-time QR code generation
- Clean and responsive UI
- Easy to use interface
- Lightweight project (no backend required)

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- QR Code API (or library like QRServer / qrcode.js)
